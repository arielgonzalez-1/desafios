# Convertidor de sistemas numericos

## Instrucciones:
1. Clonar el repositorio:
    ``` bash
    git clone https://github.com/c05m4r/desafios.git
    ```
2. Instalar colorama con pip3:
    ``` bash
    pip install colorama
    ```
3. Ejecutar el programa:
    ``` bash
    python main.py
    ```

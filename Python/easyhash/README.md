# Easyhash

## Instrucciones:
1. Clonar el repositorio:
    ``` bash
    git clone https://github.com/c05m4r/easyhash.git
    ```
2. Instalar Tkinter (en Fedora):
    ``` bash
    sudo dnf install python3-tkinter
    ```
3. Ejecutar el programa:
    ``` bash
    python main.py
    ```
